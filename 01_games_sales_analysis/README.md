# Исследование рынка компьютерных игр («Стримчик»)

## Описание

Интернет-магазин «Стримчик» продаёт компьютерные игры по всему миру.
Из открытых источников доступны исторические данные о продажах игр,
оценки пользователей и экспертов, жанры и платформы.
Нужно выявить определяющие успех игры закономерности, чтобы
спланировать рекламную кампанию на следующий год.

## Цель

Провести исследовательский анализ данных и проверить статистические
гипотезы, которые помогут бизнесу сделать ставку на наиболее
перспективные платформы и жанры.

## Данные

`games.csv` — продажи игр по регионам (NA, EU, JP, Other),
год релиза, платформа, жанр, оценки критиков и пользователей,
возрастной рейтинг ESRB.

## Что сделано

- Предобработка: приведение типов, обработка пропусков и `tbd`,
  поиск неявных дубликатов.
- Анализ жизненного цикла платформ (~5 лет на пике, ~7 лет полный цикл).
- Определение актуального периода (2012–2016) для прогнозов.
- Корреляция продаж с оценками критиков / пользователей по
  ТОП платформам (рассеяние + Пирсон).
- Портреты пользователей по регионам (NA, EU, JP) по платформам,
  жанрам и рейтингу ESRB.
- Две гипотезы (t-тест Уэлча):
  - средние пользовательские рейтинги XOne и PC одинаковы — **не отвергнута**;
  - средние рейтинги жанров Action и Sports различаются — **отвергнута в пользу H1**.

## Стек

`pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy.stats`
