# Выбор локации для нефтяной скважины

## Описание

Добывающей компании «ГлавРосГосНефть» нужно решить, где бурить
новую скважину. В трёх регионах собраны пробы нефти:
качество и объём запасов измерены. Задача — построить модель
предсказания объёма запасов и на её основе выбрать регион с
наибольшей ожидаемой прибылью при контролируемом риске убытков.

## Цель

Определить регион, где разработка принесёт наибольшую прибыль при
риске убытков не более 2.5%.

## Данные

Три файла `geo_data_{0,1,2}.csv`, по 100 000 скважин в каждом:
`id`, три технических признака `f0/f1/f2` и целевая переменная
`product` (объём запасов, тыс. баррелей).

## Что сделано

- Обучение линейной регрессии отдельно для каждого региона, оценка RMSE и
  среднего предсказанного объёма.
- Расчёт безубыточного объёма скважины (~111.1 тыс. баррелей).
- Функция прибыли: выбор 200 лучших скважин из 500 разведанных
  и расчёт выручки.
- **Bootstrap** (1000 подвыборок) — 95%-й доверительный интервал прибыли
  и вероятность убытков для каждого региона.

## Результат

Выбран **Регион 1**: средняя прибыль ≈ 456 млн ₽, 95% ДИ [34 млн; 852 млн],
риск убытков 1.5% (< 2.5%). Регионы 0 и 2 имеют риск убытков 6.9% и 7.6%
соответственно — не подходят под требование заказчика.

## Стек

`pandas`, `numpy`, `sklearn.linear_model`, `sklearn.model_selection`,
`sklearn.metrics`, Bootstrap
