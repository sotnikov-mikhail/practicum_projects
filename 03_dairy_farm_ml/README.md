# Модель закупки коров для фермы «Вольный луг»

## Описание

Молочное хозяйство «Вольный луг» планирует покупку новых коров
у ассоциации «ЭкоФерма». Для коров из своего стада известны
характеристики, удой и вкус молока; для кандидатов на покупку —
только характеристики. Нужно построить две модели: предсказать
удой и вкус молока по характеристикам, чтобы принимать решение
о покупке.

## Цель

1. **Регрессия:** прогноз удоя в литрах за год.
2. **Классификация:** прогноз «вкусное / не вкусное» молоко
   с ограничением `Precision ≥ 0.95` (важно не купить корову
   с невкусным молоком).

## Данные

- `ferma_main.csv` — характеристики текущего стада + удой;
- `ferma_dad.csv` — признаки отцов коров текущего стада;
- `cow_buy.csv` — характеристики коров на покупку.

## Что сделано

- Предобработка: типы, пропуски, дубликаты, аномалии.
- EDA целевых переменных и признаков, boxplot'ы по категориям.
- Корреляционный анализ (Пирсон + **PhiK** для смешанных признаков).
- Линейная регрессия с `OneHotEncoder` / `StandardScaler`
  через пайплайн, оценка MAE / R².
- Логистическая регрессия для задачи классификации;
  подбор порога вероятности по PR-кривой для удовлетворения
  требования `Precision ≥ 0.95`.
- Применение моделей к кандидатам на покупку и формирование
  рекомендаций.

## Стек

`pandas`, `numpy`, `matplotlib`, `seaborn`, `phik`,
`sklearn` (`Pipeline`, `ColumnTransformer`, `LinearRegression`,
`LogisticRegression`, `precision_recall_curve`)
