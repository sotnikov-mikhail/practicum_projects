# Определение рыночной стоимости автомобилей («Не бит, не крашен»)

## Описание

Сервис по продаже подержанных авто разрабатывает приложение,
в котором клиент быстро узнаёт рыночную стоимость своей
машины. Нужна модель регрессии, которая предсказывает цену
по техническим характеристикам и истории автомобиля —
при этом важны и качество, и скорость предсказания / обучения.

## Цель

Построить модель, минимизирующую RMSE, и сравнить модели
по качеству, времени обучения и времени предсказания.

## Данные

`autos.csv` — анкеты о проданных авто: дата анкеты, тип кузова,
пробег, марка/модель, мощность, тип коробки, тип топлива,
ремонт, год регистрации и др., цена продажи.

## Что сделано

- Предобработка: аномалии по году и мощности, редкие категории,
  заполнение пропусков (в т.ч. категория `unknown` для типа кузова),
  удаление признаков с мультиколлинеарностью / утечками.
- Корреляционный анализ через `PhiK` для смешанных признаков.
- Обучение и сравнение моделей:
  - линейная регрессия,
  - случайный лес,
  - **LightGBM** с подбором гиперпараметров.
- Замер времени обучения и инференса для каждой модели.
- Выбор лучшей модели по соотношению качества и скорости,
  финальное тестирование на hold-out.

## Стек

`pandas`, `matplotlib`, `seaborn`, `phik`, `sklearn`, `lightgbm`, `time`
