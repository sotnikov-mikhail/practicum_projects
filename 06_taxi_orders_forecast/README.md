# Прогнозирование заказов такси

## Описание

Компания такси собрала исторические данные о заказах в аэропортах.
Чтобы привлекать больше водителей в часы пик, нужна модель,
прогнозирующая количество заказов такси на следующий час.

## Цель

Построить модель, предсказывающую количество заказов на следующий
час, с метрикой RMSE ≤ 48 на тесте.

## Данные

`taxi.csv` — ряд заказов с 10-минутной дискретизацией
(ресэмплирован до часа).

## Что сделано

- Ресэмплирование ряда до часа, визуализация.
- Декомпозиция временного ряда (`statsmodels.seasonal_decompose`):
  тренд, сезонность, остатки; суточная и недельная сезонность.
- Генерация признаков: календарные (`hour`, `dayofweek`, `month`),
  лаги, скользящее среднее.
- Обучение и валидация моделей на `TimeSeriesSplit`:
  линейная регрессия, случайный лес, градиентный бустинг.
- Финальное тестирование лучшей модели на hold-out.

## Стек

`pandas`, `numpy`, `matplotlib`, `statsmodels`, `sklearn`
(`TimeSeriesSplit`, `LinearRegression`, ensembles)
