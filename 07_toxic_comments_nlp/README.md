# Классификация токсичных комментариев

## Описание

Интернет-магазин запускает новый сервис: пользователи могут
редактировать описания товаров и предлагать правки. Нужен
инструмент, который автоматически находит токсичные комментарии
и отправляет их на модерацию.

## Цель

Обучить модель классификации текстов с метрикой **F1 ≥ 0.75**
на тесте.

## Данные

`toxic_comments.csv` — англоязычные комментарии с бинарной меткой
`toxic`.

## Что сделано

- Предобработка текста: очистка, лемматизация через `spaCy`.
- Сравнение трёх подходов к векторизации:
  - **Bag of Words**,
  - **TF-IDF**,
  - **Word2Vec** (`gensim`).
- Обучение нескольких моделей-классификаторов
  (логрег, дерево, RandomForest и др.) поверх каждой векторизации.
- Подбор гиперпараметров `Pipeline + GridSearchCV` на лучшей
  векторизации, финальное тестирование.

## Стек

`pandas`, `numpy`, `nltk`, `spacy`, `gensim`, `sklearn`,
`matplotlib`, `wordcloud`, `tqdm`
