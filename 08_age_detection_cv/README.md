# Определение возраста покупателей по фотографии

## Описание

Сетевой супермаркет внедряет систему компьютерного зрения для
работы с изображениями покупателей на кассах: оценка возраста
нужна для анализа покупок в разных возрастных группах и для
контроля продажи алкоголя.

## Цель

Обучить модель, предсказывающую возраст человека по фото,
с **MAE ≤ 8** на тесте.

## Данные

~7591 фотография лиц с разметкой возраста (`labels.csv`,
папка `final_files/`). Диапазон возрастов 1–100, пик —
молодые взрослые 15–40 лет.

## Что сделано

- EDA: распределение возраста (гистограмма + boxplot),
  всплески в «юбилейные» годы (20/25/30 и т.д.) как индикатор
  шума разметки.
- Просмотр образцов изображений, описание особенностей
  (ч/б фото, повороты, закрытые лица).
- Аугментации: `horizontal_flip`, `rotation`, `brightness`,
  `zoom`.
- Архитектура — **ResNet50** (ImageNet) с разморозкой слоёв,
  `GlobalAveragePooling2D` → `Dense(1, relu)`.
- Обучение: `Adam(lr=1e-4)`, MSE-лосс, 10 эпох.
- Итоговое качество: **val_MAE ≈ 6.05** — цель достигнута.

## Стек

`pandas`, `numpy`, `matplotlib`, `tensorflow.keras`
(`ImageDataGenerator`, `ResNet50`, `GlobalAveragePooling2D`)
