# Персонализация предложений для интернет-магазина «В один клик»

## Описание

Интернет-магазин «В один клик» хочет удержать постоянных
клиентов. Задача — предсказать вероятность снижения
покупательской активности и сегментировать клиентов,
чтобы маркетологи могли точечно работать с каждой группой.

## Цель

Построить модель классификации, которая прогнозирует снижение
активности у постоянных покупателей, и на её основе выделить
сегменты для персональных предложений.

## Данные

- `market_file.csv` — активность клиента в онлайн-магазине;
- `market_money.csv` — выручка по клиенту;
- `market_time.csv` — время на сайте;
- `money.csv` — средняя прибыль на клиента.

## Что сделано

- Предобработка и объединение четырёх датафреймов.
- EDA и корреляционный анализ признаков.
- **Пайплайн**: `ColumnTransformer` + `StandardScaler` + `OneHot` + модель.
- Сравнение моделей классификации
  (логрег, случайный лес, бустинг, SVC) через `GridSearchCV`.
- Анализ важности признаков (`SHAP`).
- Сегментация клиентов: выделение группы с высоким риском
  снижения активности и высокой маржинальностью, формулировка
  рекомендаций для маркетинга.

## Стек

`pandas`, `numpy`, `matplotlib`, `seaborn`, `statsmodels`,
`patsy`, `sklearn` (`Pipeline`, `ColumnTransformer`, `GridSearchCV`),
`shap`
