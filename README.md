# Practicum projects

Учебные проекты, выполненные во время курса
**«Специалист по Data Science»** от Яндекс.Практикума.

Каждый проект — отдельная папка с README и тетрадью
Jupyter в `notebook/`.

## Список проектов

| № | Проект | Тема | Стек |
|---|---|---|---|
| 01 | [Исследование рынка игр «Стримчик»](01_games_sales_analysis) | EDA, проверка гипотез | `pandas`, `numpy`, `matplotlib`, `seaborn`, `scipy` |
| 02 | [Выбор локации для скважины](02_oil_well_location) | Линейная регрессия, **Bootstrap**, бизнес-метрики | `pandas`, `numpy`, `sklearn` |
| 03 | [Модель закупки коров «Вольный луг»](03_dairy_farm_ml) | Регрессия + классификация с ограничением по Precision | `sklearn`, `phik`, `seaborn` |
| 04 | [HR-аналитика «Работа с заботой»](04_hr_analytics) | Регрессия удовлетворённости + классификация увольнения | `sklearn`, `phik`, SMAPE / ROC-AUC |
| 05 | [Стоимость авто «Не бит, не крашен»](05_car_price_prediction) | Градиентный бустинг, сравнение по скорости | `sklearn`, `lightgbm`, `phik` |
| 06 | [Прогноз заказов такси](06_taxi_orders_forecast) | Временные ряды, сезонность, лаги | `statsmodels`, `sklearn`, `TimeSeriesSplit` |
| 07 | [Токсичные комментарии](07_toxic_comments_nlp) | NLP, BoW / TF-IDF / Word2Vec | `nltk`, `spacy`, `gensim`, `sklearn` |
| 08 | [Определение возраста по фото](08_age_detection_cv) | Computer Vision, ResNet50, transfer learning | `tensorflow.keras` |
| 09 | [Персонализация «В один клик»](09_ecommerce_personalization) | Пайплайны, `SHAP`, сегментация клиентов | `sklearn`, `shap`, `statsmodels` |
| 10 | [Отток клиентов «ТелеДом»](10_telecom_churn) | Финальный проект: объединение данных, бустинг, ROC-AUC | `sklearn`, `phik` |
